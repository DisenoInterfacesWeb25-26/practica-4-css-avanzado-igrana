¿Cómo se adapta esta web?

Crear una maqueta de una web que modifique su composición dependiendo del dispositivo con el que se visualice. Utilizar para ello un diseño responsive con media queries, no se puede utilizar Flexbox o CSS Grid. La maqueta se guardará en dos archivos llamados maqueta.html y maqueta.css.

Pantalla de ordenador (a partir de 1200px)

El tipo de letra para el tamaño de pantalla de ordenador será Arial.

Pantalla de tablet (a partir de 700px y hasta 1200px)

El tipo de letra para el tamaño de pantalla de tablet será Calibri.

Pantalla de móvil (hasta 700px)

El tipo de letra para el tamaño de pantalla de móvil será Times New Roman.
